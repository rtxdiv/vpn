Как купить подписку

Через приложение

Откройте приложение.

Выберите подходящий тариф и период оплаты.

Создайте платёж и перейдите в платежи.

Выполните перевод по указанным реквизитам.

Обязательно добавьте указанное сообщение к переводу без изменений!

Переводы рассматриваются вручную в течение 24 часов. Не паникуйте при долгой обработке – администратор, вероятно, спит

При возникновении вопросов вы можете обратиться в поддержку

Как получить конфигурацию

Скопировать ссылку

Откройте приложение.

На карточке купленной подписки нажмите на иконку шестерёнки.

Нажмите на кнопку "Скопировать ссылку" – она автоматически добавится в буфер обмена

Открыть QR-код

Откройте приложение.

На карточке купленной подписки нажмите на иконку шестерёнки.

Нажмите на кнопку "Подписка"

Как начать использование

Android / iOS

Скопируйте ссылкуна конфигурацию или откройте QR-код

Из магазина приложений скачайте v2RayTun или Happ.

Нажмите кнопку добавления подписки ("+" сверху справа) и выберите Import from clipboard (импорт из буфера обмена) или Import from QR (отсканировать QR-код).

После добавления нажмите на кнопку запуска по центру

Windows

Установите v2RayTun с официального сайта.

Нажмите кнопку добавления подписки ("+" сверху справа) и выберите Import from clipboard (импорт из буфера обмена) или Import from QR (отсканировать QR-код).

После добавления нажмите на кнопку запуска по центру

Linux

Установите Hiddify-Hext из официального GitHub репозитория для вашего дистрибутива.

Нажмите кнопку добавления подписки ("+" сверху справа) и выберите импорт из буфера обмена.

После добавления нажмите на кнопку запуска по центру
